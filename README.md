# 🌙 Driftveil — A Sleep Game

**Driftveil** is an interactive sleep experience designed to gently guide you into rest. It's not a game you win — it's a game you fall asleep to.

## ✨ Features

- 🌀 **Breathing Orb** — Expands and contracts to guide your breathing, gradually slowing over time
- ✨ **120 Drifting Particles** — Soft glowing lights that pulse and respond to your touch
- 👆 **Touch Trails** — Glowing paths that fade like light through water
- 🎧 **Generative Ambient Audio** — Layered drones, ocean-like noise, and subtle delta-wave binaural beats
- 🌑 **Auto Sleep Mode** — World fades after 30s of inactivity, reaching near-darkness at 15 min
- 🧘 **Mind Empty Mode** — Breathing cycle stretches from 6s → 12s over time
- 💤 **No score. No timer. No failure.** — Just drifting.

## 🎮 How to Play

1. Open `index.html` in any browser (or visit the GitHub Pages link)
2. Tap/click anywhere to begin
3. Move your finger or mouse slowly across the screen
4. Breathe with the central orb
5. Let go...

## 🎧 Best Experience

- Use **headphones** for binaural beat effect
- Go **fullscreen** (F11 on desktop)
- **Dim your room**

## 🧠 Design Philosophy

> The goal isn't to entertain — it's to gently occupy the mind just enough that it stops racing… and then disappears.

Based on sleep psychology principles:
- Repetitive motion + visual feedback = meditative state
- Slow breathing synchronization
- Progressive reduction of stimulation
- "Let go" over "take control"

## 🛠 Tech

Pure HTML5 / Canvas / Web Audio API — no dependencies, no build step, works offline.
